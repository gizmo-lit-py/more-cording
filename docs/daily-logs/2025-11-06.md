今日やったこと：サプーの動画でzip,enumerate,tqdm　等のfor文で使うと便利な関数をした
学びメモ：二つを同時に使うこともできる


コード：

items = ['りんご','みかん','バナナ','ぶどう']
stocks = [5,0,2,8]

for i, (item, stock) in enumerate(zip(items,stocks), start=1):
    if stock == 0:
        print(f'{i},{item}は在庫がありません。')
    else:
        print(f'{i},{item}の在庫は{stock}個です。')
